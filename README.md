# Python Revisit

A structured revision of core Python concepts — from variables and data types to object-oriented programming. Built as a quick reference for refreshing fundamentals.

## Topics Covered

| Module | Concepts |
|--------|----------|
| **Variables & Basics** | Numeric, Boolean, None, String, List, Set, Tuple, Dict types |
| **Operators** | Arithmetic, Comparison, Logical, Membership operators |
| **Conditionals & Control Flow** | `if` / `elif` / `else` with real-world scenarios |
| **Loops** | `while`, `for`, `range()`, `continue`, `break`, `pass` |
| **Lists & Dictionaries** | Reference vs copy, `get()`, `keys()`, `values()`, `items()`, `pop()` |
| **OOP Concepts** | Classes, constructors, polymorphism with inheritance |

## Usage

Run any file with Python 3:

```bash
python "Variables & Basics.py"
```

## Purpose

This repo serves as a personal reference for Python syntax and concepts, organized by topic for quick lookup and revision.
